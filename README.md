# Mixxx-Tools

This package contains a collection of tools for Mixxx :in the folder Mixxx Scripts Tiers you find many tools that i found on internet for Mixxx. I added a few others in folder Scripts Divers. The main contribution here is a tool in Python capable of handling Mixxx Data Base in order to transport it to another computer. This proves usefull if you re working on different coputers amsd want to update them. It updates Mixxx playlists, colors, beatgrids, hotcues, crates.
You can update specific data or all data, using command lines described in User Manual.
